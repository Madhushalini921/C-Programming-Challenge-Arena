# C Programming Challenge Arena

C Programming Challenge Arena is a console-based mini game hub developed using the C programming language. It contains 10 interactive games designed to improve programming and problem-solving skills while providing an enjoyable user experience.

## Games Included

- C Programming Quiz
- Guess The Number
- Rock Paper Scissors
- Tic Tac Toe
- Superhero Rescue Mission
- Hangman
- Snake and Ladders
- Treasure Hunt Adventure
- Castle Escape
- Space Adventure

## Technologies Used

- C Programming
- GCC Compiler
- Modular Programming

## How to Run

Compile the project using:

```bash
gcc *.c -o arena
```

Run the executable:

```bash
arena.exe
```
